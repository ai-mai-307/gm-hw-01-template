# Домашнее задание №1 — Генеративные модели

## Описание задания
В этом домашнем задании вам предстоит обучить и сравнить три класса генеративных моделей: **Energy-Based Model (EBM)**, **Variational Autoencoder (VAE)** и **диффузионную модель (DDPM / score-based)**. Все модели должны научиться генерировать **произвольные изображения котов** по датасету Kaggle "Cats Image (64*64)"

## Постановка задачи и начисляемые по ним баллы

### Подзадача 1. Подготовка данных (общая для всех моделей): 1 балл

1. Реализовать [pytorch обертку](https://docs.pytorch.org/tutorials/beginner/basics/data_tutorial.html) над датасетом (класс ```class CatsDataset(Dataset)``` 
2. Выполнить предобработку изображений (например, передавая в ```CatsDataset``` инстанс класса ```torch.nn.Sequential```, внутри которого находятся последовательность алгоритмов предобработки из [torchvision](https://docs.pytorch.org/vision/0.8/transforms.html)). Она должна включать, как минимум нормализацию значения пикселей;
3. Реализовать `DataLoader` для train и val подмножеств данных;
4. Функцию визуализации изображений из батча.


### Подзадача 2. Energy-Based Model (EBM): 3 балла
1. Реализовать функцию энергии `E_θ(x)` в виде нейронной сети(например, CNN, выдающая скаляр энергии).
2. Выполнить обучение  модели
3. Продемонстрировать качества обучения с помощью:
    - графиков функции потерь для train и val
    - визуализацией нескольких сгенерированных изображений

### Подзадача 3. Variational Autoencoder (VAE): 2 балла
1. Реализовать VAE в виде нейронной сети
2. Выполнить обучение  модели
3. Продемонстрировать качества обучения с помощью:
    - графиков функции потерь для train и val
    - визуализацией нескольких сгенерированных изображений

### Подзадача 4. Диффузионная модель (DDPM): 2 балла
1. Реализовать диффузионную модель
2. Выполнить обучение  модели
3. Продемонстрировать качества обучения с помощью:
    - графиков функции потерь для train и val
    - визуализацией нескольких сгенерированных изображений

### Подзадача 5. Сравнение моделей и отчёт: 1 балл
1. Сравнить три подхода по:
   - визуальному качеству генерации;
   - скорости генерации;
   - устойчивости обучения и сложности реализации.
2. Привести короткое рассуждение: что оказалось проще/сложнее, какая модель дает стабильные и правдоподобные генерации
